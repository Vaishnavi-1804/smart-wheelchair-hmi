# Smart Wheelchair (HMI)

This project implements a smart wheelchair using EMG signals and gesture control.

## Features
- EMG signal acquisition using ESP32
- Gesture control using MPU6050
- Real-time data logging using Python

## Structure
hardware/ → ESP32 code  
software/ → Python scripts  
data/ → stored EMG data  

## Status
Currently capturing and analyzing EMG signals.
